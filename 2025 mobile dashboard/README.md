# 2025 Global Mobile Phones Analysis

An interactive **Power BI Dashboard** analyzing the 2025 smartphone market, featuring specifications, pricing, and performance trends across major brands.

## 📋 Project Overview

This project explores the projected 2025 global smartphone market with detailed analysis of pricing, hardware specifications, 5G support, and brand positioning.

### Key Objectives:
- Analyze pricing trends across brands and segments
- Compare hardware specifications (RAM, Storage, Camera, Battery)
- Evaluate 5G adoption and OS distribution
- Identify top-performing models and value-for-money options

## 🛠 Technologies Used

- **Power BI Desktop** (Interactive Dashboard)
- **DAX** (Calculations & Measures)
- **Power Query** (Data Transformation)
- **Excel** (Data Source)

## 📊 Dataset

- **File**: `Global_Mobile_Prices_2025_Extended.csv`
- **Rows**: ~800+ projected 2025 models
- **Key Columns**: Brand, Model, Price (USD), RAM (GB), Storage (GB), Camera (MP), Battery (mAh), Display Size, Charging Speed, 5G Support, OS, Processor, Rating

## 📈 Dashboard Features

- **Market Overview**: Total models, average price, 5G adoption rate
- **Brand Comparison**: Price distribution and average specs per brand
- **Performance Analysis**: RAM, Storage, Camera, and Battery insights
- **Value Analysis**: Best phones by price-to-performance ratio
- **5G & OS Trends**: Adoption rates and operating system breakdown
