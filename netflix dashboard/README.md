An interactive **Power BI Dashboard** analyzing Netflix's content library, trends, and performance metrics using a cleaned dataset of over 8,800 titles.

## 📋 Project Overview

This project explores Netflix's vast content catalog to uncover insights about movies vs TV shows, content distribution by country, genre popularity, rating trends, and more.

### Key Objectives:
- Analyze content trends over time
- Understand geographic distribution of content
- Identify top genres, directors, and actors
- Create an interactive business intelligence dashboard

## 🛠 Technologies Used

- **Power BI** (for interactive dashboards)
- **Excel** (data cleaning & preparation)
- **DAX** (for calculations and measures)
- **Power Query** (for data transformation)

## 📊 Dataset

- **Source**: Netflix Titles Dataset
- **Rows**: 8,807 titles
- **Columns**: show_id, type, title, director, cast, country, date_added, release_year, rating, duration, listed_in, description

**Cleaned Version**: `netflix_titles_cleaned.xlsx`

## 📈 Dashboard Features

- **Overview Page**: Total content, Movies vs TV Shows split
- **Content Trends**: Growth over years, release year distribution
- **Geographic Analysis**: Top countries producing content
- **Genre Insights**: Most popular categories and sub-genres
- **Ratings & Duration**: Average ratings and content length analysis
- **Top Directors & Actors**: Most prolific creators on Netflix
